# CustomerSuite

## Purpose
Customer relationship management with calendar and document generation.

## Local Development
```bash
cd /home/yousef/Documents/workspace/zonevast/apps/CustomerSuite
npm install
npm run dev -p 3008
```
Access at: http://localhost:3008

## Key Features
- Customer profiles and management
- Calendar integration (FullCalendar)
- Document/PDF generation
- Customer interactions tracking
- Address management
- Customer analytics
- Loyalty programs

## Tech Stack
- Next.js 14 (App Router)
- FullCalendar (scheduling)
- PDF generation libraries
- @tanstack/react-query
- @nextui-org/react
- next-intl
- GraphQL
