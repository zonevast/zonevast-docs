# InventorySuite

## Purpose
Inventory and stock management with 3D visualization for plant tracking.

## Local Development
```bash
cd /home/yousef/Documents/workspace/zonevast/apps/InventorySuite
npm install
npm run dev -p 3003
```
Access at: http://localhost:3003

## Key Features
- Stock management and tracking
- Multi-location inventory
- Plant tracking with 3D view (Three.js)
- Low stock alerts
- Inventory movements (IN/OUT/TRANSFER)
- Batch and expiration tracking
- Tax calculations

## Tech Stack
- Next.js 14 (App Router)
- Three.js (3D visualization)
- @tanstack/react-query
- @nextui-org/react
- next-intl
- GraphQL
