# BlogSuite

## Purpose
Content management system for blog posts and articles.

## Key Features
- Blog post creation and editing
- Rich text editor (TinyMCE)
- Content publishing
- Media management
- Categories and tags
- SEO optimization
- Scheduled publishing

## Tech Stack
- Next.js 14 (App Router)
- TinyMCE (rich text editor)
- @tanstack/react-query
- @nextui-org/react
- next-intl
- GraphQL
